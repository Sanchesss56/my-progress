Добро пожаловать!
Для синхронизации между устройствами нужен GitHub токен и Gist ID. Вводятся один раз и сохраняются в браузере.
1
Создай токен на github.com/settings/tokens/new — галочка только на gist, срок — No expiration
2
Создай Gist на gist.github.com — файл назови progress-data.json, содержимое [], нажми Create secret gist
3
Скопируй ID из адресной строки: gist.github.com/юзер/вот_это
Хранится только в вашем браузере, не в коде
